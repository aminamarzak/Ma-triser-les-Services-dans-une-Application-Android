# Ma-triser-les-Services-dans-une-Application-Android

Objectifs du lab
Créer une application Chronomètre Service entièrement en Java qui utilise :
Un Foreground Service (obligatoire depuis Android 8.0)
Une notification persistante qui montre le temps en direct
Un Bound Service pour communiquer avec l’Activity
Démarrage/arrêt depuis l’interface
Vous comprendrez parfaitement le cycle de vie des Services, onStartCommand, onBind, START_STICKY, les restrictions Android et les bonnes pratiques.
